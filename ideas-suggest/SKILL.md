# ideas-suggest

## Procedure

1. Read `.ai/PROJECT_MAP.md` (if not exists → scan file_tree manually)
2. Analyze across 5 categories: 🚀 Perf | 🏗️ Arch | 🎨 UX/DX | 🔒 Security | 📦 Code
3. **Max 10 ideas**. Each idea:
   - Описание (1-2 предложения)
   - Какую проблему решает
   - Усилие: 🟢 < 1ч | 🟡 1-4ч | 🔴 > 4ч
   - Конкретный пример кода или метрика эффекта
4. Sort by impact/effort (quick wins first)
5. Output:
```markdown
## 💡 Идеи
### Quick Wins (🟢)
| # | Идея | Эффект | Категория |
### Средние (🟡)
| # | Идея | Эффект | Категория |
### Стратегические (🔴)
| # | Идея | Эффект | Категория |
---
### 1. <Название>
**Сейчас:** 3 запроса к БД на каждый вызов API
**Предложение:** объединить в JOIN
**Эффект:** latency 150ms → 50ms
```
6. Ask: какие реализуем? → create plan for chosen.
